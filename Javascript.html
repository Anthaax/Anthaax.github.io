JavaScript


Le JavaScript est un langage de programation de script. Orienté objet le JavaScript utilise le principe de prototypage. Un prototype est un objet à partir duquel on peu crée de nouveau objets.

Le JavaScript fut le premier langage de programation que j'ai utilisé pour apprendre à developper durant ma formation. J'ai appris grace à NODE.js.

J'apprecie le JavaScript par sa simplicité de comprehenssion mais aussi par la puissance du langage qui permet de voir un peu la programation fonctionnele.

Ce ne fut pas tout le temps le cas au debut l'aspect ultra modulable ne me plaisait pas trop car sur des gros projets cela devien vite illisible et immatenable.
Mais maintenant avec plus de recul et une plus grande rigueur dans ma façon de develloper j'apprecie plus JavaScript qu'auparavant.

De plus l'apporche fonctionnele du langage est très interessante elle permet de façon plutot sure et sans trop d'erreurs à la fin.

Durant mes deux années chez NOTOCORD dans le projet SENSE j'ai utilisé le JavaScript très souvent l'on utilise AngularJS pour la partie web de l'application.
Apres le C# c'est le langages de programtion que je maîtrise le mieux.
