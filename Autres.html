Dans cette partie je vais évoquer les autres compétences acquises durant ma formation et mon travail chez NOTOCORD.

ASP.NET : ASP.NET est un Framework permettant de générer des pages web et des traitements de requêtes. J’utilise ASP.NET tous les jours au sein du projet SENSE. Je l’ai aussi utilisé dans le projet ARCHI’VITE. Cela crée une facilité de communication entre le serveur et l’application web.

Windows Forms : Durant le développement du projet Simc’ITI nous avons utilisé Windows Forms pour l’interface utilisateur. La facilité de création de fenêtre grâce au système de « drag and drop » est appréciable. C’est aussi grâce à Windows Forms que j’ai le plus utilisé la notion d'événements dans un projet.

Xamarin : J’ai utilisé Xamarin dans le projet ARCHI’VITE pour faire l’interface sur smartphone. J’ai essayé d’utiliser les bonnes façons de faire pour créer une application. Je n’ai acquis que les bases de l'outil.

Unity3D : Durant le projet Synouk Must Die, l’interface graphique fut développée sous Unity3D. J'ai passé peu de temps sur cet outil, j’ai travaillé sur la création des personnages et la gestion des combats.
