Autres


Dans cette partie je vais parler des autres compétences que j’ai acquise durant ma formation et mon travail chez NOTOCORD.

ASP.NET : ASP.NET est un Framework permettant de générer des pages web et traitement de requêtes. J’utilise ASP.NET tous les jours au sein du projets SENSE. Je l’ai aussi utilisé dans le projet Archi’vite. J’apprécie la facilité avec laquelle on peut discuter avec le serveur et l’application web.

Windows Forms : Durant le développement du projet Simc’ITI nous avons utilisé Windows Forms pour l’interface utilisateur. La facilité de création de fenêtre grâce au système de « drag and drop ». C’est aussi grâce à Windows Forms que j’ai le plus utilisé les événements dans un projets.

Xamarin : J’ai utilisé Xamarin dans le projet Archi’vite pour faire l’interface sur smartphone. J’ai essayé d’utiliser les bonnes façons de faire pour créer une application. Je n’ai que la bases sur Xamarin vu que je ne l’ai utilisé qu’une seule fois et de façon pas trop poussé.

Unity3D : Durant le projet Synouk Must Die l’interface graphique fut développé sous Unity3D. Je n’ai pas passé énormément de temps dessus. J’ai travaillé sur la création des personnages et la gestion des combats.
