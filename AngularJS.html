AngularJS


AngularJS est un Framework Javascript libre. Il facilite la création d'application web côté front.

J'utilise AngularJs depuis que je travaille chez NOTOCORD. Je n'avais jamais utilisé de Framework Javascript côté front auparavant.


Un nouveau Framework à découvrir

La facilité avec laquelle il est possible de construire des pages dynamiquement grâce à ce Framework, est très agreable. Moi qui n'aimais pas trop le developpement web, j'ai apprecié cette facilité.
J'apprécie tout particulièrement le principe de "component". Cela permet la création simple de site web avec l'ajout d'éléments génériques.
Grace à AngularJS, j'ai pu comprendre la différence fondamentale en MVC et MVVM, ainsi que les erreurs à ne pas commettre comme le "two way binding" qui peut être certes efficace mais créatrice d'incohérences lors du développement d’un gros projet. Chez NOTOCORD, l'équipe de développement compte passer sur Angular4 pour la partie web à la place de AngularJS. Angular4 est aussi un Framework facilitant la création de pages web dynamiques. Cela nous fera découvrir plus en profondeur le Typescript. TypeScript est un langage de programmation libre et open source développé par Microsoft qui a pour but d'améliorer et de sécuriser la production de code JavaScript. Le TypeScript a été cocréé avec le principal inventeur de C#.

AngularJS m'a permis d'oublier certains a priori que j'avais sur le développement web, j'ai trouvé ce mode de développement beaucoup plus "propre" que le PHP. J'ai fait aussi de la transmission de connaissances sur l'AngularJS au sein de l'équipe Sense.

J'utilise AngularJS depuis que je travaille au sein de l'équipe SENSE. D'ailleurs, je propose maintenant à l'équipe, des améliorations sur l'application en utilisant le plus efficacement AngularJS.
Je pense être très familier avec ce Framework.
