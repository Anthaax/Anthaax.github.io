C#


Le C# est un langage fortement typé orienté objet et destiné à être développer sur la plateforme .NET.

Durant la deuxième année de ma formation à In’tech nous avons appris à utiliser les langages objets pour le développement et le langage choisit pour apprendre fût le C# J'ai tout de suite adoré ce langages de programation. Depuis je l'utilise pour tous mes projets étudiants, professionnel et personnel.

J’ai put essayer le JAVA aussi durant ma formation mais j’ai préféré utiliser le C# car de base j’ai préféré l’IDE Visual studio par rapport aux autres utilisé en JAVA.

De plus maintenant avec le Framework .NET CORE permet de faire des applications cross plateform, je ne m’intéresse plus trop au JAVA.
Et me fait penser que le C# va s'imposer comme le langage objet n°1.

Ce que j’apprécie dans les langages objets les concepts d’architecture pour créer une application. Depuis le début j’essaie de faire attention à la hiérarchie des différentes classes pour faciliter l’ajout de fonctionnalité.

J’ai utilisé le C# dans différents types applications :

  • Un jeu vidéo de gestion : Simc’Iti
  • Une application d’aide à la personne : Archi'Vite
  • Un jeu vidéo Rogue-like : Synouk Must Die
  • Une application web : SENSE

Le C# est le langage que je préfère le plus utilisé et celui ou je suis le plus qualifié. Cela fait 4 ans que j’utilise le C# tous les jours pour développer.
