C#


Le C# est un langage fortement typé orienté objet et destiné à être développé sur la plateforme Microsoft .NET. Il est commercialisé par Microsoft depuis 2002.

Durant la deuxième année de ma formation à In’tech, nous avons appris à utiliser les langages objets pour le développement et le langage choisi fût le C# J'ai tout de suite adoré ce langage de programation. Depuis je l'utilise pour tous mes projets étudiants, professionnels et personnels.

Ce que j’apprécie dans les langages objets, ce sont les concepts d’architecture pour concevoir une application. Depuis le début j’essaie d'être attentif à la hiérarchie des différentes classes pour faciliter l’ajout de fonctionnalités.

J’ai utilisé le C# dans différents types d'applications :

  • Un jeu vidéo de gestion : Simc’Iti
  • Une application d’aide à la personne : Archi'Vite
  • Un jeu vidéo Rogue-like : Synouk Must Die
  • Une application web : SENSE


Une valeur sûre

Le C# est maintenant dans le monde du developpement depuis plus de 15 ans. Il a su évoluer avec son temps et maintenant il est l'un des plus utilisé par les développeurs. Contrairement à d'autres langages qui sont très utilisés car ils sont éphémères car dans le buzz du moment, le C# lui, est stable dans son évolution et n'est pas près de disparaitre. J’ai également pu essayer le JAVA durant ma formation mais j’ai préféré utiliser le C# car j’ai préféré l’IDE Visual studio par rapport aux autres disponibles en JAVA.
Maintenant, le Framework .NET CORE permet de faire des applications cross plateform, je ne m’intéresse moins au JAVA.
Il est probable que le C# dépasse JAVA en utilisation grace à ce nouveau Framework d'ici quelques années.

Le C# est le langage que je préfère et pour lequel j'ai le plus de compétences. Je l’utilise depuis 4 ans tous les jours dans mon activité de développeur.
